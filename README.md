# Song Ranker Skeleton

Elo-based ranker for arbitrary collections.

## Setup
Local server required.
Example: `python3 -m http.server`

## Usage
1. Populate `song_data.js`.
2. Configure `config.js`.
3. Open `index.html`.

## Features
- Elo ranking system
- Custom list support
- Playlist export
- Multiple collection support
- Basic Supabase integration
